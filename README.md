# SynthV-Types

SynthV-Types は JavaScript で Synthesizer V Studio のスクリプトを作成するための型定義コレクションです。

## Usage

スクリプトディレクトリに [`synthv.d.ts`](./synthv.d.ts) を配置し、以下のコードをスクリプトの先頭に追加してください。

```js
/// <reference path="./synthv.d.ts" />
```

これにより、Synthesizer V Studio の API 補完を利用したり、型チェックを行ったりすることができます。
